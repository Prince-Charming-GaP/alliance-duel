## ðŸ’¡ Key Concepts

**State Ruler** is a 2-3 week cross-server tournament where servers compete for the prestigious State Ruler title. Success requires:

- **Preparation Phase discipline** - Daily event participation from Sunday-Friday determines Invader/Defender roles
- **Saturday coordination** - 4-hour Capital Clash battle determines tournament winners
- **Hospital management** - Avoiding the critical 30% permanent troop death on overflow
- **Resource stockpiling** - Sustained combat requires deep resource reserves

**Critical Principle:** Coordination trumps raw power. Well-organized servers regularly defeat higher-power opponents through strategic planning and tactical execution.
